Manufacturing Lot No.

Easy generation of internal reference for a product variant.

Why Manufacturing Lot No.?

Let me first give you an overview of this module.
This module will allow you to create custom Lot/Serial Number for all Product SKUs, which will provide easy traceability. It is difficult to identify a product from the lot/serial number that the system generates as it is just a sequence that is no different for unlike products. We have taken this point into consideration and have found a solution to it.

To make your work easier, we have brought this app that will create a customized lot/serial number with the prefix you have mentioned. This will become easier to distinguish between different products. It is traceable too!

You might have not thought of, but this minute feature is of great significance.

Detailed Features

  • Allows you to customize Lot/Serial Numbers.
  • You must set an internal reference in order to generate customized serial numbers.
  • This allows you to decide a prefix of your choice.
  • All the lot/serial numbers of that product will have the given prefix followed by 4 digit sequence.
  • It becomes easy to identify the product just from its lot/serial number.
  • It provides easy traceability from the manufacturing order.
  • It saves time required for identifying the product from its lot/serial number generated by the system.
To begin with, have the Manufacturing Lot No. Module downloaded in your system. For its proper setup, go through the below steps.

Manufacturing Lot No. Module
Firstly, you need to install the Manufacturing Lot No. App from Odoo Apps. It's easy to locate, just search on the name, you can identify it with the manufacturing icon. Hold on, you also need to enable it.

Apps >> Manufacturing Lot No. >> Install

Enable this feature
In order to enable this feature, go to general settings, select Inventory, under Traceability, activate "Lots & Serial Numbers".

General Settings >> Inventory >> Traceability >> activate "Lots & Serial Numbers"

Now you are ready to utilise this feature. For more information go through the Work Flow.

Create a Product Variant
Move to product variants, create a new product variant. Fill in the name, make the Product Type as Storable Product, then add an Internal Reference in order to customize your serial number. Do not forget setting Tracking to "By Unique Serial Number" before saving the form.
It will automatically generate the serial number.

Inventory >> Products >> Product Variants >> Create

Serial No.
Once you save the product variant, a sequence will be generated automatically. You can see that in the Serial No. page of the product variant form. The sequence is traceable, isn't it? Click on it, and have a look.

New Product Variant >> Serial No. >> Sequence

Bill of Material
On the same form, you would find Bill of Material present among the buttons at the top. Click on it and create a Bill of Material for the new product variant. You only need to fill a component on that page.
Well, our BOM is also ready now!

New Product Variant >> Bill of Material >> Create >> Add a component

Create a Manufacturing Order
Now, move to Manufacturing app, and create an order for the newly created product variant. In that, you just need to fill the product name. Do not forget to save and confirm it.

Apps >> Manufacturing(MRP) >> Create >> Fill Product name >> Save and Confirm

Create a new serial/lot number
As soon as you confirm the manufacturing order, you will see the Lot/Serial Number field. To create new serial/lot number, click on the plus icon. A reference is created.

Confirmed Manufacturing Order >> Lot/Serial Number >> click on Plus(+) icon

Reference Created
On clicking the plus icon, a new lot/serial number is created, and its reference is saved in that field. Isn't that amazing?
