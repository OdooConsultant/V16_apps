Sales Order From File

Easy order directly from file with product barcodes

Why Sales Order From File?

Let me first give you an overview of this module.
This module will create a sales order directly from an excel file containing list of barcodes in a specific format. It is very tedious work to manually add product lines for a huge order, especially when hundreds/thousands of products are to be ordered. We have taken this point into consideration and have found a solution for it.

In order to lessen your work, we have brought this app that will generate a sales order from a list of barcodes mentioned in an excel file. This will save you a lot of time, which you can utilize to enhance your business instead.

There is one more perk of using this app. It allows you to enter same barcode multiple times, and would automatically calculate the total quantity, just like that.

Hot Features

Generates sales order from excel file
Only one column for barcodes is required in excel file
Allows multiple entries of same barcode
Calculates total quantity for same barcodes
Also displays unmatched barcodes
Saves a lot of time


Detailed Features

  • Generates a sales order directly from an excel file.
  • The excel file needs to have only a column with list of barcodes.
  • It will allow multiple entries of same barcode.
  • You do not need to worry about the total quantity, it calculates it for you.
  • It lessens your work of manually adding products in the product lines.
  • Works good with a sales order of hundreds/thousands of products.
  • It even reflects you with the unmatched barcodes of the excel file.
  • This makes it easier for you to compare the barcodes.
To begin with, have the Sales Order From File downloaded in your system. For its proper setup, go through the below steps.

Sales Order From File Module
Firstly, you need to install the Sales Order From File App from Odoo Apps. It's easy to locate, just search on the name, you can identify it with the sale quotation form icon. Ta-da, you are ready to go!!

Apps >> Sales Order From File >> Install

Excel File
Create an excel file as shown above with correct product barcodes. Make sure that the File which you want to import for Sales Order contains barcodes in the format Alphabet/Number.
You can add same barcodes multiple times. You do not need to worry about total quantities, as this app will generate the total for same barcodes automatically.

Create Excel File >> Enter Barcodes

Sales Order From File Location
You might wonder how could a Sales order be created from an excel file. But, we have made it possible!

Now, open Sales App, under Order you would find Sales Order From File in the drop-down menu. Click on that.
Apps >> Sales >> Order >> Sales Order From File

Create Order Wizard
Well, if you are stuck, here is a small guide on filling the form.

For the customer field, select a customer for whom you want to create the sales order and then you just need to upload the excel file with the list of barcodes. Do not forget to click on Create Order button at the end.

Sales Order
Once, you click on Create Order button, a new sales order is generated. In that new sales order, you see only those products whose barcodes matched with that of the excel file.
Now, that's amazing, isn't it?

New Sales Order >> Product lines

Import Logs
It's not over yet! There is one more thing you might want to know.
Well, there can also be a case where in few barcodes of the excel file wouldn't find a match. You can inspect the list of those barcodes in the Import Logs page of the sales order.

Sales order >> Import Logs
