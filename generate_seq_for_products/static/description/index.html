Generate Product Sequence

Easy generation of internal reference of product variant.

Why Generate Product Sequence?

Let me first give you an overview of this module.
This module will allow you to create SKUs even for the variants of the products. So, a product having let's say 4 variants, will have unique code for each variant. It now becomes easy to find the product variant directly with its SKU. The task of locating a specific variant of a product is lengthy, you first need to search for the product and then check out its variants.

Moreover, if the name of the product changes, it becomes even more difficult. The introduction of our module has made this task much easier. You now directly have to search with the product variant's SKU or the Internal Reference. Our app generated SKUs for all the product variants. Further, it can also be utilized to generate barcodes.

So, it saves your precious time that can be utilized to enhance your business instead. You might have not thought of, but this minute feature is of great significance.

Detailed Features

  • Allows you to enable the IN Product Reference.
  • You can add codes to the various product attributes.
  • It will automatically generate SKUs or Internal References for the product Variants.
  • You can also add SKUs for products without variants.
  • It allows easy search for the specific variant of a product with SKU.
  • It saves time required for locating the product first and then the variant.
  • Further, it can also be used to generate barcodes.
To begin with, have the Generate Product Sequence Module downloaded in your system. For its proper setup, go through the below steps.

Generate Product Sequence Module
Firstly, you need to install the Generate Product Sequence App from Odoo Apps. It's easy to locate, just search on the name, you can identify it with the sequence icon. Hold on, you also need to enable it.

Apps >> Generate Product Sequence >> Install

Enable this feature
Go to Inventory or Sales, under Configuration select Attribute, activate IN Product Reference for any attribute. Now, you will be able to access this wonderful feature. Do not forget to add code for attributes.

Inventory/Sales >> Configuration >> Attributes >> activate IN Product Reference >> Add code for attributes

Now you are ready to utilise this feature. For more information go through the Work Flow.

Create a new Product
Move to products, create a new product. In variants, add the attribute for which you enabled IN Product Reference and save it.
Sales >> Products >> Products >> Create >> Variants >> Add attribute

It will automatically generate product SKU/Internal Reference. You can see that by locating the product variant.
Sales >> Products >> Product Variants >> Select the product variant just created

Product SKU
We can see the evaluation of the code for each variant of the product. The Internal reference looks like "Product UID + Attribute Code".

It can also generate product SKU for single as well as multiple products even without variants.
Products >> select the newly created product >> Click the variant button present on top
