Quick Operations in SO

Allows salesperson to validate deliveries, validation of invoice, and registering cash payment from SO itself at busy sales counters

Why Quick Operations in SO?

Let me first give you an overview of this module.
This module will give you the direct access for delivery validation, invoice validation and for registering payment directly from the sales order form. In busy sales counters, it becomes lengthy task to validate deliveries and invoices from different tabs. In order to avoid this prolonged task, we provide you with an app that can do everything within the order form.

Once the sales order is confirmed, it will show you a button to validate delivery, after which it will even allow you to create and validate invoices at the same place. It also allows you to register payment on the same window.

So, the customer need not to wait long for the completion of the whole process. There is one more perk of using this app. Once the payment is registered, it will open the invoice automatically. What you need to do is just print that invoice and hand it over to the customer. That's it!

To begin with, have the Quick Operations in SO Module downloaded in your system. For its proper setup, go through the below steps.

Quick Operations in SO Module
Firstly, you need to install the Quick Operations in SO App from Odoo Apps. It's easy to locate, just search on the name, you can identify it with the all-in-one icon. Ta-da, you are ready to go!!

Apps >> Quick Operations in SO >> Install

Create a Sales Order
Open Sales app, tap on create and fill the basic fields. Do not forget to add products. Save and confirm it.

Apps >> Sales >> Create >> Fill in the form with few products >> Save >> Confirm

Delivery Validate
To validate the delivery, you do not have to move to the delivery page. We provide a direct button in the sales order form. Click on Delivery Validate button. A wizard will pop up.

Confirmed Sales Order >> Delivery Validate

Apply for done quantities
The wizard will have two buttons, click on the apply button. This means that the order is fulfilled. Hold on, you are almost done!

Confirmed Sales Order >> Delivery Validate >> Apply

Create Invoice
In order to generate an invoice, click on Create Invoice button. A wizard to create invoice will pop up.

Confirmed Sales Order >> Delivery Validate >> Create Invoice

Create Invoice Wizard
You will have to select the invoice type i.e. Regular, Down Payment (Percentage) or Down Payment (Fixed Amount). Fill in the other requirements if any. Click on the Invoice button at the end.

Confirmed Sales Order >> Delivery Validate >> Create Invoice >> select Invoice Type >> Create Invoice

Validate Invoice
To validate the sale order invoice, click on Validate Invoice button on the top. Have patience, just two more steps to go!

Confirmed Sales Order >> Delivery Validate >> Create Invoice >> Validate Invoice

Register Payment
To register the payment of the sales order, tap on Register Payment button. A Register Payment Wizard will show up.

Confirmed Sales Order >> Delivery Validate >> Create Invoice >> Validate Invoice >> Register Payment

Create Payment button
Fill in the necessary details required for the payment and do not forget to tap on Create Payment button. Ta-da, your payment is done too!!

Confirmed Sales Order >> Delivery Validate >> Create Invoice >> Validate Invoice >> Register Payment >> Create Payment

Customer Invoice
You can now access paid invoice from the Invoice button on the top-right corner.
